Cấu hình file config.js bao gồm các thông số
- folder bin trong thư mục cài đặt chương trình matlab
- folder chứa source code matlab trong chương trình
- folder upload image
- folder dùng để ghi log file kết quả thực thi của module malab
Tải đường dẫn https://drive.google.com/file/d/0B1qy5I115cOFMVdVSEJtYWRZZ3c/view?usp=sharing và giải nén vào thư mục matlab/result để nạp bộ dữ liệu đã được train dùng cho việc tìm kiếm ảnh.
Cài đặt NodeJS, mở CMD và dùng lệnh cd để đi đến thư mục /project, tại đây gõ lệnh “npm install” để cài đặt các pagekages cần thiết cho node server, sau lệnh npm install, chạy lệnh “gulp” để build các file source và start web server.
Truy cập web từ địa chỉ localhost:3000 để sử dụng.
